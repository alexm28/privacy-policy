# Privacy Policy for TelegramContent

**Last updated:** April 30, 2026

## Overview
This application ("TelegramContent") is a personal tool that connects to the Strava API to read my own athletic activities and generate running posts via AI.

## What data is collected
- Strava activity data (distance, time, pace, type, date)
- No data is collected from other users
- No personal identifying information is stored

## How data is used
- Data is used exclusively to generate personal posts
- Posts are sent to my personal Telegram account
- Comments are added to my own Strava activities

## Data sharing
- No data is shared with any third parties
- No data is sold or transferred

## Data retention
- Activity IDs are stored locally to prevent duplicate posts
- No long-term database storage is used

## Your rights
- You can revoke this application's access at any time via Strava settings
- All stored data will be deleted upon access revocation
